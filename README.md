# CoachMate (코치메이트)

운동전문가를 위한 AI 플랫폼

**업무는 더 스마트하게, 코칭은 더 깊게**

## 구조

```
CoachMate
├── Smarter (더 스마트한 업무)
│   ├── 운동일지 AI
│   └── 식단관리 AI (준비 중)
└── Better (더 깊은 코칭)
    ├── 교육 (준비 중)
    ├── 자격증 (준비 중)
    └── 채용 (준비 중)
```

## 파일

| 파일 | 설명 |
|------|------|
| `index.html` | 메인 |
| `smarter.html` | Smarter 허브 |
| `coaching-log.html` | 운동일지 AI |
